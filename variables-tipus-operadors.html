🧱 Nivell 1 — Variables, Tipus i Operadors

Fonaments de C# per a Videojocs

1.1 — Què és una variable?

És un espai de memòria RAM amb nom que guarda un valor. Quan declares int vida = 100; passen 4 coses:

  • Es reserva un espai de 32 bits a la RAM.
  • Es defineix que l'espai és de tipus int.
  • S'associa el nom vida a la posició.
  • S'escriu el valor 100.
int vida = 100;

⚠️ float vs double:

La GPU prefereix 32 bits. No oblidis la 'f'.

float velocitat = 3.5f;

1.7 — Operadors lògics

&& (AND)

Les dues condicions han de ser certes.

if(vida > 0 && mana > 10)

|| (OR)

N'hi ha prou que una sigui certa.

if(vida <= 0 || temps <= 0)

! (NOT)

Inverteix el valor booleà.

if(!estaViu)

Un sistema de decisió complet basat en l'estat:

if (vida < 30 && !estaEnCobertura) {
    Console.WriteLine("Fugir!");

    Console.WriteLine("Curar-se");
