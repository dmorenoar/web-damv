🧠 Nivell 2 — Estructures de Control

La lògica que pren decisions i controla el flux

1. Estructures de control (decisió)

Permeten que el joc prengui camins diferents segons les circumstàncies.

🟢 if: Cas Únic

int health = 30;
if (health <= 0) {
    Console.WriteLine("Game Over");
}

🎮 Si la vida arriba a 0 → el jugador mor.

🟡 if - else: Dues alternatives

int ammo = 0;
if (ammo > 0) {
    Console.WriteLine("Dispares!");
} else {
    Console.WriteLine("No tens munició!");
}

🎮 Si tens bales → dispares. Si no → has de recarregar.

🟠 if - else if - else: Múltiples estats

int score = 750;
if (score >= 1000) {
    Console.WriteLine("Rang: Or");
} else if (score >= 500) {
    Console.WriteLine("Rang: Plata");
} else {
    Console.WriteLine("Rang: Bronze");
}

2. Switch: Estats Discrets 

int option = 2;
switch (option) {
    case 1: Console.WriteLine("Nova partida"); break;
    case 2: Console.WriteLine("Carregar partida"); break;
    default: Console.WriteLine("Opció no vàlida"); break;
}

for

for(int i=1; i<=3; i++)

while

Mentre sigui cert.

while(lives > 0)

do-while

Almenys un cop.

do{...}while(c);
